impl PeerRequest

A future which wraps an APIMessage REQUEST->RESPONSE into a Future(e.g. REQBLOCK->RESULT__). This enables a much cleaner interface for inter-node message relays by allowing a response to be returned directly from peer.send_command() and to turn an ERROR___ response into an Err() Result.
