Expand description

The Mempool holds unprocessed blocks and transactions and is in control of discerning when the node is allowed to create a block. It bundles the block and sends it to the Blockchain to be added to the longest-chain. New Blocks received over the network are queued in the Mempool before being added to the Blockchain
